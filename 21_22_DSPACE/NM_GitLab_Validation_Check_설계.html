Project Name

  • 텍스트 검사
    • 문자, 숫자, 이모티콘
    • 특수문자 : _ - .
    • 특수문자 _ 로 시작 가능하다.
    • 공백 2글자 이상일 경우 하나의 1글자로 변경
  • 길이 검사
    • 최대 255글자
    • 최소 1글자
  • 텍스트 중복 검사
    • 대소문자 구분 O QWE = qwe 같다
    • 최상위 프로젝트에서 중복된 프로젝트 이름은 불가능하다

Project Slug

  • 텍스트 검사
    • 문자, 숫자
    • 특수문자 : _ - .
    • - 를 제외한 다른 특수문자는 맨 앞에 넣을 수 있다.
    • . .git .atom 으로 끝날 수 없다.
  • 길이 검사
    • ID 와 / 를 포함해서 최대 255글자
    • 최소 1글자
  • 텍스트 중복 검사
    • 대소문자 구분 X QWE ≠ qwe 다르다
    • 최상위 프로젝트에서 중복된 프로젝트 주소는 불가능하다
## 길이 검사 설명
- ID : 91298184
- Slug : MINSEOK
- Project Path : https://gitlab.tbdspace.kt.co.kr/91298184/MINSEOK
https://gitlab.tbdspace.kt.co.kr/ 를 제외한 
"91298184/MINSEOK" 가 최대 255글자까지 

D-Space 에서는 사번 8글자를 사용하기 때문에
사번8글자 + /1글자 = 9글자 
255 - 9 = 246 
Project Slug 는 최대 246글자를 사용할 수 있다.
